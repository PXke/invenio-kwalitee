Installation
============
Invenio-Kwalitee is on PyPI so all you need is: ::

    pip install Invenio-Kwalitee

Testing
=======
Running the tests are as simple as: ::

    python setup.py test

or (to also show test coverage) ::

    source run-tests.sh
